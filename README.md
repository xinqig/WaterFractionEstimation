Python program for surface water fraction estimation using Gaussian Process Regressing with Landsat and MODIS products. 

The model is developed for the publication: Surface Water Fraction Estimation Using Satellite Data for Flood Monitoring.

Satellite products that are needed to run the scripts:

Water Observations: https://thredds.nci.org.au/thredds/catalog/jw04/ga_ls_wo_3/catalog.html

MODIS: https://ladsweb.modaps.eosdis.nasa.gov/archive/allData/61/MOD09GA/

Landsat: https://earthexplorer.usgs.gov/
